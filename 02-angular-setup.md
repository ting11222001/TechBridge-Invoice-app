Install Angular CLI:
https://angular.dev/installation
```
npm install -g @angular/cli
```

Create a new angular project:
```
ng new <project name>
```

Start up the angular app:
```
cd <project root folder>
ng serve
```

Create a login component:
```
ng generate c component/login --skip-tests=true
```
